## 摘要
脑电图（EEG）基础模型越来越依赖于多数据集训练和评估，然而公开的EEG数据集仍然缺乏一个共享的任务规范层，该层可以将异构记录转化为可复用的基准单元。现有标准组织了文件、元数据和来源，但并未在通用语言和规则手册下指定EEG任务，导致关键任务语义分散在论文、代码和人工解释中。我们研究了是否可以通过结构化的任务规范语言与共享规则手册相结合，对异构的公开EEG数据集进行标准化。我们的方法将每个基准条目表示为一个与可执行任务内核同步的任务文档，规则手册定义了任务字段、证据要求、文档-内核对齐、审查状态和机器可检查的约束。使用这种方法，我们发布了一个经过社区审查的EEG基准语料库，包含53个已完成和审查的条目，涵盖245个跨多种范式的任务定义，并引入了NeuroDoc和NeuroAudit作为操作支持层，用于基于规则手册的起草、升级、审查、修订和发布管理。我们进一步研究了生成的基准单元是否可以在四个EEG基础模型骨干的共享下游设置中实例化，为可复用、可审计和可执行的EEG基准测试基础设施提供了基于执行的证据。

## Abstract
Electroencephalography (EEG) foundation models increasingly rely on multi-dataset training and evaluation, yet public EEG datasets still lack a shared task specification layer that can turn heterogeneous recordings into reusable benchmark units. Existing standards organize files, metadata, and provenance, but they do not specify EEG tasks under a common language and rulebook, leaving critical task semantics scattered across papers, code, and manual interpretation. We investigate whether heterogeneous public EEG datasets can be standardized through a structured task specification language paired with a shared rulebook. Our methodology represents each benchmark entry as a task document synchronized with an executable task kernel, with the rulebook defining task fields, evidence requirements, document-kernel alignment, review states, and machine-checkable constraints. Using this methodology, we release a community-reviewed EEG benchmark corpus centered on 53 completed and reviewed entries with 245 task definitions spanning diverse paradigms, and we introduce NeuroDoc and NeuroAudit as the operational support layer for rulebook-guided drafting, upgrading, review, amendment, and release management. We further examine whether the resulting benchmark units can be instantiated in a shared downstream setting across four EEG foundation model backbones, providing execution-based evidence for reusable, auditable, and executable EEG benchmarking infrastructure.

### 2. 论文提出的方法论

- **核心思想**：提出一个**基于规则手册（Rulebook-Guided）的EEG任务规范语言**，将每个异构的公开EEG数据集转化为一个可复用的“基准单元”（Benchmark Entry）。
- **关键技术细节**：
    - **基准单元（Benchmark Entry）**：每个条目由两个同步的部分组成：

        2.  **任务内核（Task Kernel）**：可执行的代码，将文档中的规范实例化为一个统一的EEG处理流水线，确保任务可以被一致地执行。
    - **任务规范语言**：将任务分解为四个耦合的层：
        1.  **任务类型（Task Type）**：分类或回归。
        2.  **Epoch方法（Epoch Method）**：事件锁定、长事件锁定或连续窗口。
        3.  **标签路径（Label Route）**：事件（event）、杂项（misc）或信息（info），对应MNE库中的不同数据源。
        4.  **元级别（Meta Level）**：原始元（raw meta，直接从数据读取）或超级元（super meta，需要进一步推导）。
    - **规则手册（Rulebook）**：治理基础，定义了可接受的证据、任务声明的合法性、文档-内核对齐、审查状态（如`completed`, `hallucination`）和机器可检查的约束。它确保了构建过程的标准化和可审计性。

- **充分性与客观性**：
    - **充分性**：实验覆盖了多种主流EEG范式和模型，验证了框架的通用性。通过CS和WS两种评估模式，提供了更全面的性能视角。实验还展示了学习曲线、难度梯度等，分析较为深入。
    - **客观性**：论文明确指出，其定量分析仅限于53个“已完成并审查”的条目，其余约200个草稿条目未被纳入，这保证了分析基础的可靠性。实验设计（如使用Macro-F1作为主要指标）是合理的。然而，论文并未进行消融实验来量化NeuroDoc/NeuroAudit工具链本身带来的具体增益（例如，与纯人工构建相比的效率提升），这是一个潜在的不足。

### 6. 论文的主要结论与发现

- **主要结论**：提出的基于规则手册的任务规范语言和方法论能够有效地将异构的公开EEG数据集转化为显式、可执行、可审计的基准单元。
- **关键发现**：
    - 构建的基准语料库具有广泛的任务覆盖和结构多样性，支持多任务、多标签路径和多Epoch方法。
    - 下游验证实验表明，这些标准化的基准单元可以被四个不同的EEG基础模型成功消费，并产生有意义的、可解释的任务级输出。
    - 实验揭示了跨被试（CS）和被试内（WS）评估之间存在显著的性能差距（平均12.6个百分点），强调了评估协议的重要性。
    - NeuroDoc能够生成高质量的草稿，但质量高度依赖于数据集家族（如OpenNeuro/BIDS表现最好，PhysioNet和TUH较弱），表明LLM辅助构建仍有局限性，需要人工审查。

### 7. 优点

- **问题定义清晰**：精准地识别了EEG基准测试领域一个被忽视但至关重要的瓶颈——任务规范缺失。
- **方法论系统且严谨**：提出的“规则手册-文档-内核”三层架构设计精巧，将抽象的任务概念具体化、标准化和可执行化。规则手册的引入确保了构建过程的规范性和可审计性。
- **工具链完整**：NeuroDoc和NeuroAudit的配套开发，使得方法论不仅停留在理论层面，而是形成了一个可操作的、支持社区协作的完整工作流。
- **资源贡献巨大**：发布的53个经过社区审查的基准条目是一个宝贵的公共资源，将极大促进未来EEG研究的可复现性和公平比较。
- **验证方式务实**：通过让四个主流EEG基础模型作为“消费者”来执行这些任务，有力地证明了其框架的实用性和有效性。

### 8. 不足与局限

- **实验覆盖有限**：定量分析严格限制在53个“已完成”条目，而更大的数据库（约200个草稿条目）未被纳入，这限制了结论的普适性。这些草稿条目的质量和构建效率是未知的。
- **缺乏消融研究**：论文没有量化NeuroDoc工具链（特别是LLM辅助）相对于纯人工构建的效率提升或质量差异。其声称的“LLM辅助”优势缺乏直接的实验证据。
- **LLM依赖风险**：NeuroDoc的生成质量高度依赖LLM，且在不同数据集家族上表现差异巨大（如PhysioNet和TUH表现不佳）。这暗示了LLM在处理复杂、非标准化的临床数据时存在局限性，可能引入幻觉或错误。
- **社区治理挑战**：虽然提出了规则手册和社区审查流程，但论文并未深入探讨如何在实际中维持一个活跃、高效、公正的社区审查机制。这将是该框架长期成功的关键挑战。
- **伦理与隐私风险**：论文在附录G中讨论了风险，但核心框架本身并未内置隐私保护机制。虽然不重新分发原始数据，但标准化的任务定义可能使针对敏感生物医学数据的攻击或推断变得更容易。
