## 摘要
轻度认知障碍（MCI）的早期可扩展检测仍是一个未解决的临床挑战。现有的基于脑电图的筛查方法受限于手工特征流程，这些流程丢弃了具有神经生理学意义的结构信息，而深度学习分类器则以牺牲可解释性为代价追求性能。目前尚无研究将生理学组织的概念编码器、跨概念交互建模以及非线性表格分类统一于基于睡眠脑电图的MCI检测框架中。本研究提出了概念引导的多项式变换表格学习结合Kolmogorov-Arnold网络（CPTabKAN），该方法将异构的脑电图衍生特征映射为领域知情概念表示，通过二阶多项式变换进行扩展以暴露一阶和二阶交互，并应用傅里叶参数化的TabKAN分类器学习非线性决策边界。CPTabKAN在骨质疏松性骨折研究队列（372名受试者，整夜多导睡眠图）上进行了评估，使用了按十个生理动机概念组组织的1,379个特征。在10折交叉验证下，CPTabKAN-二阶实现了加权F1分数0.9038（标准差0.034），比梯度提升高出5.65个百分点（t(9)=1.934, p=0.043，单侧配对检验），在基于SMOTE的平衡下优势依然存在。消融分析证实了每个组件的独立贡献。概念重要性分析显示，功率谱密度、多尺度熵和Hjorth参数主导了第一阶权重，而涉及Lempel-Ziv-Welch复杂度、统计量、人口统计学和慢振荡的跨概念交互则超过了所有第一阶分数。这些结果表明，概念结构化、交互感知的表格学习能够呈现生理上连贯的推理，支持临床信任。

## Abstract
Early and scalable detection of mild cognitive impairment (MCI) remains an unresolved clinical challenge. Existing EEG-based screening approaches are constrained by handcrafted feature pipelines that discard neurophysiologically meaningful domain structure and deep learning classifiers that sacrifice interpretability for performance. No existing work unifies physiologically organized concept encoders, cross-concept interaction modeling, and nonlinear tabular classification in a sleep EEG-based MCI detection framework. This study proposes Concept-guided Polynomial-transformed Tabular learning using Kolmogorov-Arnold Network (CPTabKAN), which maps heterogeneous EEG-derived features into domain-informed concept representations, expands them via degree-2 polynomial transformation to expose first- and second-order interactions, and applies a Fourier-parameterized TabKAN classifier to learn nonlinear decision boundaries. CPTabKAN was evaluated on the Study of Osteoporotic Fractures cohort (372 subjects, overnight polysomnography), using 1,379 features organized into ten physiologically motivated concept groups. Under 10-fold cross-validation, CPTabKAN-Second Order achieved a weighted F1-score of 0.9038 (SD 0.034), outperforming GradientBoosting by 5.65 percentage points (t(9)=1.934,p=0.043, one-sided paired test), with advantages persisting under SMOTE-based balancing. Ablation analysis confirmed independent contributions from each component. Concept importance analysis revealed that power spectral density, multi-scale entropy, and Hjorth parameters dominated first-order weights, while cross-concept interactions involving Lempel-Ziv-Welch complexity, statistics, demographics, and slow oscillations exceeded all first-order scores. These results demonstrate that concept-structured, interaction-aware tabular learning surfaces physiologically coherent reasoning, supporting clinical trust.

### 论文结构化总结

#### 1. 核心问题与整体含义（研究动机与背景）

- **研究问题**：早期、可扩展的轻度认知障碍（MCI）检测是一个临床难题。现有的基于睡眠脑电图（EEG）的方法存在三大局限性：
    1.  **结构缺失**：传统方法将异构的EEG特征（如频谱、熵、人口统计等）拼接成一个扁平向量，丢弃了特征背后的神经生理学组织结构。
    2.  **可解释性差**：深度学习方法（如LSTM、CNN）性能虽好，但作为“黑箱”模型，缺乏决策过程的透明度，这在需要临床信任的医疗场景中是关键障碍。
    3.  **交互缺失**：现有框架未能显式分析和建模不同生理特征域之间的交互作用（如睡眠复杂度与慢振荡形态之间的关系），而这些交互可能包含重要的判别信息。
- **整体含义**：该研究旨在填补一个空白——即没有一个现成的框架能同时实现以下三点：保留生理概念结构、使跨概念交互可显式分析、并在该结构化空间内进行灵活的非线性分类。其核心动机是，通过引入可解释的生理知识归纳偏置，来提升MCI检测的**性能**与**可信度**。

#### 2. 方法论：核心思想与关键技术

- **核心思想**：提出一个名为**CPTabKAN**的框架，该框架由三个核心模块顺序组成，将可解释的生理知识融入深度学习流程。
    1.  **概念表示模块**：
        - **输入**：将1,379维的EEG特征向量 \( x_i \) 根据生理意义划分为10个概念块。
        - **过程**：每个概念块由一个专用的概念编码器处理，输出一个标量概念分数。所有10个分数构成一个紧凑的瓶颈概念向量 \( z_i \)。
        - **关键技术**：使用带有分组卷积（组数=10）的1x1卷积层对每个概念块进行独立压缩和编码，强制保持概念级别的独立性。
        - **核心作用**：将高维、异构的特征压缩为低维、有生理含义的表示，作为“归纳偏置”。

    2.  **多项式交互展开模块**：
        - **输入**：概念向量 \( z_i \)。
        - **过程**：应用**二阶多项式特征展开** \( \phi(z_i) \)。对于10个概念，生成所有一阶项、平方项和两两交互项，共66维向量。
        - **核心作用**：将原本隐含在深度网络中的跨概念交互**显式化**为可观察、可分析的输入特征。设计上选择固定为2阶，以平衡表达能力与可解释性。

    3.  **非线性分类模块（TabKAN）**：
        - **输入**：展开后的66维概念交互向量。
        - **过程**：使用基于**Kolmogorov-Arnold网络（KAN）** 的表格分类器 **TabKAN**。与普通MLP在神经元上使用固定激活函数不同，KAN在网络**边上**使用可学习的单变量函数来拟合非线性。
        - **关键技术**：边缘函数采用**傅里叶级数（FourierKAN）** 参数化（优于B样条B-spline），共两层（第一层64神经元，grid=6；第二层512神经元，grid=7）。
        - **核心作用**：提供强大的非线性拟合能力，同时保持边缘级别的可解释性。

- **训练目标**：端到端训练，损失函数为交叉熵损失加上一个针对概念向量 \( z \) 的L2范数惩罚项（防止概念表示膨胀）。

#### 3. 实验设计

- **数据集**：

    - **数据详情**：372名老年女性受试者（平均年龄82.7岁），进行整夜多导睡眠图（PSG）监测。
    - **标签定义**：MCI阳性标签基于MMSE分数（<27分），导致约7.9:1的类别不平衡（42 MCI vs 330 正常）。
    - **特征**：设计了10个生理概念组，共提取了1,379个特征。

- **基准方法/对比模型**：
    - **传统方法**：SVM, KNN, 随机森林, 逻辑回归。
    - **集成方法**：XGBoost, LightGBM, Gradient Boosting, 极端随机树。

#### 6. 主要结论与发现
1.  **性能领先**：CPTabKAN-Second Order在主要设定下取得最优加权F1分数（0.9038），比最强基线Gradient Boosting（0.8473）高出5.65个百分点，且差异具有统计显著性（p=0.043）。
