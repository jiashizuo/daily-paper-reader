## 摘要
掩码自编码器（MAE）近期在静息态脑功能连接（FC）的自监督表征学习中展现出潜力。然而，一个基本问题仍未解决：应如何对 FC 矩阵进行标记化，使其与大尺度脑网络的内在模块化组织相一致？现有方法通常采用以脑区为中心或基于图的方案，将 FC 视为结构上同质的元素，从而忽略了大尺度脑网络组织。我们提出 NERVE（通过双线性标记化实现脑功能连接的网络感知表征），这是一种自监督学习框架，通过将 FC 矩阵划分为网络内与网络间连接块的补丁来重新定义 FC 标记化。不同于基于图像的 MAE 中固定大小的补丁共享同一标记器，由网络对定义的 FC 补丁在大小上具有异质性，并对应不同的功能角色。为解决这一问题，NERVE 通过一种新的结构化双线性分解来嵌入 FC 补丁。该公式保留了网络身份，并将参数复杂度相对于网络数量的二次缩放降低为线性缩放。我们在三个大规模发育队列（ABCD、PNC 和 CCNP）上评估了 NERVE 在行为和精神病理预测中的表现。与结构无关的 MAE 变体和基于图的自监督基线相比，所提出的网络感知公式产生了更稳定且更具可迁移性的表征，尤其是在跨队列评估中。消融研究证实，所提出的双线性网络嵌入和基于解剖学的脑区分割对性能至关重要。这些发现强调了将领域特定的结构先验纳入功能连接组学自监督学习的重要性。

## Abstract
Masked autoencoders (MAEs) have recently shown promise for self-supervised representation learning of resting-state brain functional connectivity (FC). However, a fundamental question remains unresolved: how should FC matrices be tokenized to align with the intrinsic modular organization of large-scale brain networks? Existing approaches typically adopt region-centric or graph-based schemes that treat FC as structurally homogeneous elements and overlook the large-scale network brain organization. We introduce NERVE (Network-Aware Representations of Brain Functional Connectivity via Bilinear Tokenization), a self-supervised learning framework that redefines FC tokenization by partitioning FC matrices into patches of intra- and inter-network connectivity blocks. Unlike image-based MAE, where fixed-size patches share a common tokenizer, FC patches defined by network pairs are heterogeneous in size and correspond to distinct functional roles. To resolve this problem, NERVE embeds FC patches through a novel structured bilinear factorization. This formulation preserves network identity and reduces parameter complexity from quadratic to linear scaling in the number of networks. We evaluate NERVE across three large-scale developmental cohorts (ABCD, PNC, and CCNP) for behavior and psychopathology prediction. Compared to structurally agnostic MAE variants and graph-based self-supervised baselines, the proposed network-aware formulation yields more stable and transferable representations, particularly in cross-cohort evaluation. Ablation studies confirm that the proposed bilinear network embedding and anatomically grounded parcellation are critical for performance. These findings highlight the importance of incorporating domain-specific structural priors into self-supervised learning for functional connectomics.

## 2. 方法论：NERVE 的核心思想与技术细节

### 2.1 方法总览

- 作者提出了 **NERVE**：  
  **Network-Aware Representations of Brain Functional Connectivity via Bilinear Tokenization**。

- NERVE 是一个面向 FC 矩阵的自监督 MAE 框架，其核心设计包括：
  1. **按功能网络对划分 FC 矩阵 patch**；
  2. **使用网络感知的双线性 tokenization 嵌入不同大小的网络连接块**；
  3. **在 MAE 框架下随机 mask 网络连接 token 并重建原始 FC patch**；
  4. **利用预训练模型提取 FC 表征，再用于下游行为和精神病理预测**。

---

### 2.2 网络感知 patching：将 FC 划分为网络内与网络间连接块

- 设每个被试的 FC 矩阵为：

  \[
  X \in \mathbb{R}^{R \times R}
  \]

  其中 \(R\) 是脑区数量。

- 作者使用 Schaefer 400 脑区 atlas，并根据 Yeo 功能网络将脑区分为 \(N_n\) 个大尺度功能网络。

- 对于任意两个功能网络 \(N_l\) 和 \(N_m\)，定义一个 FC patch：

  \[
  x_{l,m} \in \mathbb{R}^{|N_l| \times |N_m|}
  \]

  其中：
  - 当 \(l = m\) 时，表示网络内部连接；
  - 当 \(l < m\) 时，表示网络间连接。

- 如果共有 \(N_n\) 个功能网络，则 patch 数量为：

  \[
  N_{\text{patch}} = \frac{N_n(N_n+1)}{2}
  \]

- 在主实验中：
  - 使用 17-network parcellation；
  - 因此 \(N_n = 17\)；
  - patch 数量为 \(153\)。

---

### 2.3 为什么需要新的 tokenization？

- 由网络对定义的 FC patch 有两个特点：
  1. **尺寸不同**：不同功能网络包含的脑区数不同，因此连接块大小不同；
  2. **语义不同**：例如默认模式网络内部连接与视觉网络—注意网络连接在功能意义上不同。

- 传统图像 MAE 中常用的共享线性投影不适合直接迁移：
  - 如果对所有 patch 使用同一个投影矩阵，需要将不同大小 patch padding 到最大尺寸；
  - 这会忽略不同网络连接块的功能差异。

- 另一种做法是为每个网络对学习独立投影矩阵：
  - 这种 patch-specific linear tokenizer 表达能力更强；
  - 但参数量随网络对数量二次增长，容易过拟合，也不够可扩展。

---

### 2.4 双线性 tokenization：核心技术贡献

- NERVE 的关键创新是提出一种**结构化双线性因子分解 tokenization**。

- 对于每个功能网络 \(N_l\)，学习一个网络特异的参数矩阵：

  \[
  U_l \in \mathbb{R}^{|N_l| \times d_E}
  \]

  其中：
  - \(|N_l|\) 是网络 \(l\) 中的脑区数量；
  - \(d_E\) 是 encoder token embedding 维度。

- 对于网络对 \((l,m)\)，其投影矩阵不是独立学习，而是由两个网络参数矩阵通过 Khatri–Rao product 构造：

  \[
  W_{l,m} = U_l \odot U_m
  \]

- 对应 token 计算为：

  \[
  t_{l,m} = W_{l,m}^{\top} \text{vec}(x_{l,m})
  \]

- 直观理解：
  - 每个功能网络学习一组网络特异的 region embedding；
  - 网络间连接 patch 的 token 由两个网络 embedding 的双线性交互决定；
  - 一条连接 \((i,j)\) 对第 \(k\) 维 token 的贡献由以下乘积给出：

    \[
    [U_l]_{i,k}[U_m]_{j,k}
    \]

- 这种设计的意义是：
  - 保留功能网络身份；
  - 显式建模网络内和网络间结构；
  - 避免为每个网络对单独学习完整投影矩阵；
  - 在参数效率和表达能力之间取得折中。

---

### 2.5 参数复杂度比较

作者比较了三种 tokenization 的参数复杂度：

| 方法 | 思路 | 参数复杂度特点 |
|---|---|---|
| Shared linear | 所有 patch 共用一个线性投影 | 参数少，但忽略网络异质性 |
| Patch-specific linear | 每个网络对一个独立投影 | 表达强，但参数随网络对增长较快 |
| Bilinear tokenization | 每个网络学习参数，通过双线性组合形成网络对投影 | 参数高效，同时保留网络身份 |

在近似假设脑区均匀分布于各网络时：

- shared linear 复杂度约为：

  \[
  O\left(\frac{R^2}{N_n^2} d_E\right)
  \]

- patch-specific linear 复杂度约为：

  \[
  O(R^2 d_E)
  \]

- bilinear tokenization 复杂度约为：

  \[
  O(R d_E)
  \]

因此，NERVE 将网络对级别的参数增长转化为网络级别参数共享，实现更好的可扩展性。

---

### 2.6 MAE 自监督训练流程

NERVE 将上述网络感知 token 输入到标准 MAE 框架中：

1. 根据功能网络对划分 FC 矩阵；
2. 使用双线性 tokenizer 将每个网络连接块转为 token；
3. 加入 CLS token；
4. 加入表示网络对身份的位置编码；
5. 随机 mask 一定比例的 patch token；
6. 将可见 token 输入 Transformer encoder；
7. 在 decoder 中插入 mask token；
8. 重建被 mask 的 FC patch；
9. 以重建误差作为训练目标。

重建损失为：

\[
L_{\text{recon}} =
\frac{1}{|M|}
\sum_{(l,m)\in M}
\|\hat{x}_{l,m} - x_{l,m}\|_2^2
\]

其中 \(M\) 表示被 mask 的网络连接 patch 集合。

  - 说明该方法具有较好的跨队列迁移能力。

- 消融实验显示：
  - 双线性 tokenization 优于 shared linear 和 patch-specific linear 的整体表现；
  - 17-network parcellation 明显优于随机网络划分和固定方块 patch；
  - 当功能网络结构被破坏时，bilinear tokenization 的优势下降，说明该方法的效果依赖于有意义的功能网络先验。

- 总体结论是：
  - 将大尺度脑网络结构作为归纳偏置融入 FC 自监督学习，可以获得更稳定、更可迁移的脑连接表征。

---

## 7. 优点与亮点

- **问题定义清晰**  
  论文抓住了 FC-MAE 中一个关键但常被忽视的问题：FC 矩阵应该如何 patch/tokenize。

- **方法具有神经科学先验**  
  不是直接套用图像 MAE，而是利用大尺度功能网络组织作为结构先验，更符合脑连接数据特点。

- **双线性 tokenization 设计巧妙**  
  该方法在以下三者之间取得平衡：
  - 表达网络对差异；
  - 保留网络身份；
  - 控制参数规模。

- **参数效率高**  
  将 patch-specific tokenization 的近似 \(O(R^2 d_E)\) 参数复杂度降低到 \(O(R d_E)\)。

- **实验覆盖较全面**  
  使用三个真实大规模发育队列，并包含 out-of-domain 测试。

- **消融设计有针对性**  
  不仅比较 tokenizer，还检验了 parcellation 和真实网络结构的重要性。

- **评估方式相对稳健**  
  使用 KRR 作为统一下游 probe，避免复杂神经网络 probe 引入额外不稳定性；同时报告 bootstrap 置信区间和统计检验。

## 8. 不足与局限

- **任务和人群覆盖有限**
  - 实验集中在儿童和青少年发育队列；
  - 尚未验证成人、老年人、神经退行性疾病、精神疾病临床样本等场景。

- **下游任务较单一**
  - 主要评估行为和精神病理连续分数预测；
  - 未充分覆盖诊断分类、治疗反应预测、认知维度预测、个体识别等任务。

- **整体预测相关仍然较低**
  - 尽管 NERVE 相对更优，但多数 \(r\) 值仍处于较低到中等水平；
  - 这反映出 FC 行为预测本身难度很高，也说明实际应用仍需谨慎。

- **依赖预定义脑区分割和功能网络划分**
  - 方法建立在 Schaefer atlas 和 Yeo 网络划分之上；
  - 如果 parcellation 不准确或不适用于特定人群，可能影响性能；
  - 尚未探索个体化 parcellation 或数据驱动网络划分。

- **只使用静态 FC**
  - 当前输入是静态 FC 相关矩阵；
  - 未建模动态功能连接、时序波动或原始 BOLD 时间序列信息。

- **跨模态信息未纳入**
  - 没有结合结构连接、解剖 MRI、扩散 MRI 或临床变量；
  - 表征能力可能受限于单一模态。

- **算力报告不完整**
  - 虽然报告了使用单张 NVIDIA L40 GPU 和训练配置；
  - 但未报告训练时长、GPU hours 或超参数搜索成本。

- **baseline 公平性仍有一定限制**
  - 不同方法输入形式不完全相同；
  - 部分方法无法在 ABCD 上评估；
  - 监督模型和自监督表征模型的训练目标不同，因此比较结果需要谨慎解释。

- **可解释性尚未充分展开**
  - 作者提到未来会分析双线性 embedding 权重和 token attention；
  - 但当前论文中对模型到底学习到哪些脑网络模式的解释仍较有限。
