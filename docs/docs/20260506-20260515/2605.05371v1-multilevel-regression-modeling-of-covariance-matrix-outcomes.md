## 摘要
协方差矩阵结果在用于研究脑功能连接的神经影像实验中自然出现。理解脑网络组织如何随受试者水平协变量而变化同样具有重要意义。现有协方差回归方法采用单水平框架，无法适应受试者被分组嵌套于簇中的层级数据结构，例如生命周期研究中的年龄队列。本文提出一种多水平协变量辅助主回归（Multilevel Covariate-Assisted Principal Regression，MCAP）框架，该框架为每个簇识别一个线性投影，从而可结合协变量构建广义线性混合效应模型。簇特异性投影通过 von Mises-Fisher 分布在单位球面上建模，使得跨簇的信息借用具有原则性。模型参数通过最大化层级似然进行估计。针对推断，本文提出一种两阶段 bootstrap 程序，并建立了估计量的渐近性质。模拟研究表明，MCAP 在估计回归系数方面显著优于单水平竞争方法。将其应用于覆盖 5 至 90 岁年龄范围的人类连接组计划生命周期研究（Human Connectome Project Lifespan Study）时，MCAP 识别出一个主导的谱脑网络，刻画了年龄和性别对功能连接的影响，并揭示了若干发现，包括成年晚期神经重组模式的趋同，以及与语言和执行功能相关的跨网络区域在整个生命周期中的协调调节。

## Abstract
Covariance matrix outcomes arise naturally in neuroimaging experiments to study brain functional connectivity. It is also of interest to understand how brain network organization varies with subject-level covariates. Existing covariance regression methods operate in a single-level framework and do not accommodate the hierarchically nested data structure in which subjects are grouped into clusters, such as age cohorts in lifespan studies. A Multilevel Covariate-Assisted Principal Regression (MCAP) framework is introduced, which identifies, for each cluster, a linear projection such that a generalized linear mixed effects model can be formulated with the covariates. The cluster-specific projections are modeled on the unit sphere via a von Mises-Fisher distribution, enabling principled borrowing of information across clusters. Model parameters are estimated by maximizing a hierarchical likelihood. For inference, a two-stage bootstrap procedure is proposed. Asymptotic properties of the estimators are established. Simulation studies demonstrate that MCAP substantially outperforms single-level competitors in estimating regression coefficients. Applied to the Human Connectome Project Lifespan Study spanning ages from five to ninety, MCAP identifies a dominant spectral brain network capturing age and sex effects on functional connectivity, and reveals findings including the convergence of neural reorganization patterns in late adulthood and the coordinated lifespan modulation of cross-network regions linked to language and executive function.

## 2. 方法论：核心思想、关键技术细节与算法流程

### 2.1 核心建模思想

- 数据具有两层结构：
  - 第一级：共有 \(m\) 个簇，例如年龄组。
  - 第二级：第 \(i\) 个簇中有 \(n_i\) 个个体。
  - 每个个体 \(j\) 有 \(T_{ij}\) 个时间点的 \(p\) 维观测向量 \(y_{ijt}\)，假设：
    \[
    y_{ijt} \sim N(0, \Sigma_{ij})
    \]
  - \(\Sigma_{ij}\) 是个体 \(j\) 的协方差矩阵，即功能连接矩阵。

- MCAP 的关键假设是：
  - 对每个簇 \(i\)，存在一个单位长度投影方向 \(\gamma_i \in \mathbb{R}^p\)，使得投影后的方差：
    \[
    \gamma_i^\top \Sigma_{ij} \gamma_i
    \]
    可以通过协变量建模。
  - 也就是说，模型不直接回归整个 \(p \times p\) 协方差矩阵，而是寻找一个与协变量最相关的“协方差主方向”或“功能子网络”。

### 2.2 多水平回归模型

论文提出的主要模型为：

\[
\log(\gamma_i^\top \Sigma_{ij}\gamma_i)
=
\beta_0
+
x_{1i(j)}^\top \beta_1
+
x_{2ij}^\top(\beta_2+\vartheta_i)
+
\epsilon_i
\]

其中：

- \(\gamma_i\)：第 \(i\) 个簇的投影方向。
- \(\beta_0\)：总体截距。
- \(\beta_1\)：固定效应协变量的回归系数。
- \(\beta_2\)：随机斜率协变量的总体效应。
- \(\vartheta_i\)：簇特异随机斜率，服从：
  \[
  \vartheta_i \sim N(0,\Omega)
  \]
- \(\epsilon_i\)：簇特异随机截距，服从：
  \[
  \epsilon_i \sim N(0,\sigma^2)
  \]
- \(x_{1i(j)}\)：固定效应协变量，可以是簇水平或个体水平变量。
- \(x_{2ij}\)：随机效应协变量。

该模型实质上是在“投影后的协方差尺度”上建立广义线性混合效应模型。

### 2.3 用 von Mises-Fisher 分布建模簇特异投影方向

- 因为 \(\gamma_i\) 是单位向量，位于单位球面上，论文使用 **von Mises-Fisher 分布**建模：
  \[
  \gamma_i \sim \text{vMF}(\gamma,\kappa)
  \]
- 其中：
  - \(\gamma\)：总体平均方向。
  - \(\kappa\)：集中参数，表示各簇方向围绕总体方向的集中程度。
    - \(\kappa\) 越大，说明不同簇的投影方向越相似。
    - \(\kappa\) 越小，说明簇间方向差异更大。

这一设计使得模型既允许年龄组之间存在不同的脑网络方向，又不会完全独立估计每个年龄组，从而能实现信息借用。

### 2.4 层级似然估计

- 作者构造层级负似然函数，包括四部分：
  1. 条件于投影方向和随机效应的观测似然。
  2. 随机截距 \(\beta_{0i}\) 的正态先验 / 随机效应似然。
  3. 随机斜率 \(\beta_{2i}\) 的正态随机效应似然。
  4. 投影方向 \(\gamma_i\) 的 von Mises-Fisher 球面分布似然。

- 实际计算中，个体协方差矩阵用样本协方差：
  \[
  S_{ij} = \frac{1}{T_{ij}}\sum_t y_{ijt}y_{ijt}^\top
  \]
  替代。

- 论文选择最大化层级似然而不是边际似然，原因是：
  - 边际化掉 \(\gamma_i\)、随机截距和随机斜率后，计算非常困难。
  - 在每簇观测数增长的条件下，层级似然与边际似然具有渐近等价性。

### 2.5 投影方向的约束与数据自适应归一化

- 为避免投影方向退化，模型要求 \(\|\gamma_i\|_2=1\)。
- 计算时引入辅助向量 \(\tilde{\gamma}_i\)，并加上约束：
  \[
  \tilde{\gamma}_i^\top H_i \tilde{\gamma}_i = 1
  \]
- 其中 \(H_i\) 取为第 \(i\) 个簇的加权平均样本协方差矩阵。
- 这样做的好处是：
  - 相比简单单位球约束 \(H_i=I\)，数据自适应约束更符合样本协方差结构。
  - 更新方向时可转化为广义特征向量问题或其近似。

### 2.6 参数估计算法

论文使用 **block coordinate descent（块坐标下降）** 算法，主要步骤包括：

1. 初始化：
   - 簇方向 \(\gamma_i\)
   - 随机截距 \(\beta_{0i}\)
   - 固定效应 \(\beta_1\)
   - 随机斜率 \(\beta_{2i}\)
   - 超参数 \(\beta_0,\sigma^2,\beta_2,\Omega,\gamma,\kappa\)

2. 迭代更新：
   - 用 Newton-Raphson 更新：
     - \(\beta_{0i}\)
     - \(\beta_1\)
     - \(\beta_{2i}\)
   - 用闭式公式更新：
     - \(\beta_0\)：所有 \(\beta_{0i}\) 的均值。
     - \(\sigma^2\)：\(\beta_{0i}\) 的样本方差。
     - \(\beta_2\)：所有 \(\beta_{2i}\) 的均值。
     - \(\Omega\)：\(\beta_{2i}\) 的样本协方差。
   - 更新每个簇的投影方向 \(\gamma_i\)：
     - 解一个带椭球约束的二次优化问题。
     - 近似为广义特征向量搜索。
   - 更新总体方向和 vMF 集中参数：
     - \(\gamma\) 为所有 \(\gamma_i\) 的归一化平均方向。
     - \(\kappa\) 用 vMF 的常用近似 MLE 更新。

3. 收敛判据：
   - 当目标函数相对变化低于阈值时停止。

4. 非凸性处理：
   - 由于目标函数非凸，作者使用多次随机初始化，保留目标函数最小的解。

### 2.7 多个成分的识别

- 第一成分由 Algorithm 1 得到。
- 若要寻找第 \(k\) 个成分，论文使用类似 PCA 的 deflation 策略：
  1. 从原始数据中移除前 \(k-1\) 个已识别成分。
  2. 对残差数据做奇异值分解。
  3. 对秩亏部分进行补全。
  4. 再运行 Algorithm 1 得到第 \(k\) 个成分。

- 成分数选择使用 DfD（deviation from diagonality）指标：
  - 若投影方向能较好地同时对角化样本协方差矩阵，则 DfD 接近 1。
  - 实践中采用阈值 DfD \(\leq 2\) 判断成分数。

### 2.8 推断方法

- 论文主要对回归系数 \((\beta_0,\beta_1,\beta_2)\) 做推断。
- 对投影方向 \(\gamma_i\) 和 \(\gamma\) 不做直接推断，因为方向存在符号不确定性，且约束空间上的渐近分布更复杂。

- 提出两阶段 bootstrap：
  1. **Stage 0：全数据估计**
     - 先用完整数据估计 \(\hat{\gamma}_i\) 和其他模型参数。
     - 计算投影样本方差 \(\hat{s}_{ij}=\hat{\gamma}_i^\top S_{ij}\hat{\gamma}_i\)。
  2. **Stage 1：按簇 bootstrap**
     - 有放回抽样簇。
     - 在每个被抽中的簇内再有放回抽样个体。
     - 固定 \(\hat{\gamma}_i\)，只重新估计回归参数和方差成分。

- 这样可避免每个 bootstrap 样本中重新估计高维方向，显著降低计算成本。

### 2.9 理论性质

论文建立了两类渐近结果：

- **已知投影方向时**：
  - 回归参数具有渐近正态性。
  - 固定效应 \(\beta_1\) 的方差由 profile information matrix 控制。
  - 随机效应均值和协方差估计具有类似经典混合效应模型的渐近分布。

- **未知投影方向时**：
  - 在紧致参数空间、目标函数一致收敛、总体目标唯一最小、方向局部可识别等条件下，证明估计量一致。
  - 包括：
    - \(\hat{\gamma}_i \to \gamma_i^*\)
    - \(\hat{\gamma} \to \gamma^*\)
    - \(\hat{\kappa} \to \kappa^*\)
    - 回归参数和方差成分一致。

  - 总体每轮迭代复杂度约为：
    \[
    O(mp^3 + M_n p^2)
    \]
    其中 \(M_n=\sum_i\sum_j T_{ij}\)。
  - 作者认为该算法对中等维度数据可行，例如 \(p \leq 100\)。

- 因此，该论文更偏统计建模与数值优化，不是深度学习训练型论文，没有大规模 GPU 训练报告。

---

## 5. 实验数量与充分性

### 5.1 实验数量概况

论文实验主要包括三部分：

1. **模拟实验主结果**
   - 两个维度设置：\(p=5,20\)。
   - 两种样本规模：\(n=T=100\) 和 \(n=T=500\)。
   - 比较 MCAP 与 SCAP。
   - 报告方向相似度、bias 和 MSE。

2. **模拟推断实验**
   - 比较 bootstrap 推断与渐近正态推断。
   - 报告 95% 置信区间覆盖率。
   - 同样覆盖 \(p=5,20\) 和不同样本规模。

3. **真实数据分析**
   - HCP Lifespan Study，1,500 名受试者，17 个年龄簇，75 个 ROI。
   - 分析年龄和性别对功能连接主导子网络的影响。

### 5.2 是否充分

- **优点**：
  - 模拟实验与理论模型设定一致，能够验证方法在“正确模型”下的估计能力。
  - 同时考察低维和中等维度场景。
  - 同时比较点估计和推断性能。
  - HCP 应用规模较大，数据具有实际科学意义。

- **不足**：
  - 模拟场景相对有限。
    - 只报告了 \(p=5\) 和 \(p=20\)，而真实 HCP 分析是 \(p=75\)。
    - 没有系统考察更高维度如 \(p>100\) 或 \(p \gg n\)。
  - 对比方法较少。
    - 主要对比 SCAP。
    - 没有与其他协方差回归方法、矩阵对数回归、Cholesky 回归、Bayesian covariance regression、common principal components 等方法系统比较。
  - 消融实验较少。
    - 没有单独评估 vMF 层级建模的贡献。
    - 没有比较 \(H_i=I\) 与数据自适应 \(H_i\) 的实际差异。
    - 没有系统评估 fused lasso 稀疏化对真实数据解释的影响。
  - 真实数据只有一个主应用场景。
    - HCP 分析很有代表性，但缺乏外部神经影像数据集验证。
  - bootstrap 推断中固定 \(\hat{\gamma}_i\)，未完全传播方向估计不确定性，虽然作者给出渐近理由，但有限样本下可能偏乐观。

### 5.3 客观性与公平性

- 对比 MCAP 与 SCAP 是合理的，因为 SCAP 是最直接的单水平 CAP 基线。
- 但 benchmark 覆盖面偏窄，不足以全面证明 MCAP 优于所有主流协方差回归方法。
- 模拟数据按 MCAP 假设生成，因此对 MCAP 有一定有利性。
- 作者也承认推断方法没有纳入方向估计不确定性，这体现了一定客观性。

## 6. 主要结论与发现

### 6.1 方法学结论

- MCAP 可以有效处理层级嵌套结构下的协方差矩阵结局。
- 相比单层 SCAP，MCAP 在回归系数估计上显著更准确。
- 在维度较高或每簇样本有限时，MCAP 通过跨簇信息共享获得更稳定估计。
- bootstrap 推断通常比直接使用渐近正态近似有更好的置信区间覆盖率，尤其在有限样本下。

### 6.2 模拟实验发现

- 当 \(p=5\)、\(n=T=100\) 时：
  - SCAP 对方向估计相似度略高，但回归系数 bias 明显更大。
  - MCAP 对回归系数估计更准确。

- 当样本量增大到 \(n=T=500\) 时：
  - MCAP 的方向估计和回归系数估计均改善。

- 当维度增大到 \(p=20\) 时：
  - SCAP 的方向估计明显变差。
  - MCAP 尽管在小样本下方向估计仍有偏差，但整体优于 SCAP，且随样本量增加明显改善。

### 6.3 HCP 生命周期研究发现

- 使用 DfD 准则，模型识别出一个主导成分。
  - 说明年龄和性别相关的功能连接异质性主要集中在一个低维脑功能子网络方向上。

- 年龄效应呈非线性轨迹：
  - 投影功能连接得分从儿童和青少年期下降到 20 多岁后期。
  - 随后在中年和晚年逐渐上升。
  - 约在 70 岁左右达到峰值。
  - 70 至 90 岁再度下降。

- 性别效应：
  - 男性在该投影方向上的功能连接得分整体高于女性。
  - 估计的男性效应为 0.210，95% bootstrap CI 为 \((0.013, 0.407)\)，提示统计上显著。

- 年龄组间子网络方向：
  - 50 岁以后不同年龄簇的投影方向相似性更高。
  - 表明晚年脑网络重组可能沿着更一致的空间方向发生。
  - 儿童、青少年和早中年阶段则表现出更高的方向异质性。

- 重要脑区和网络：
  - 主要贡献区域包括：
    - rostral middle frontal gyrus, rMFG
    - middle temporal gyrus, MTG
    - inferior parietal cortex, IPC
    - isthmus cingulate gyrus, ICG
    - supramarginal gyrus, SMG
  - 涉及的功能系统包括：
    - frontoparietal network
    - default mode network, DMN
    - ventral-attention network
    - lateral temporal regions
    - visual network
    - somato-motor network
  - 这些区域与执行功能、语言、语义处理、工作记忆和高阶认知调控相关。

## 7. 优点与亮点

- **针对问题明确**：
  - 直接面向协方差矩阵结局和层级数据结构，契合神经影像生命周期研究的实际需求。

- **保留矩阵结构**：
  - 不简单向量化协方差矩阵，避免破坏正定性结构和引入过多参数。

- **低维可解释表示**：
  - 通过投影方向 \(\gamma_i\) 将复杂协方差矩阵变化压缩为可解释的子网络方差变化。
  - 结果容易与脑功能网络解释相结合。

- **允许簇间异质性**：
  - 与共享单一投影方向的 CAP 或纵向 CAP 不同，MCAP 允许每个簇有自己的协方差相关子空间。

- **跨簇信息共享合理**：
  - 使用 von Mises-Fisher 分布对单位球面上的方向建模，是处理方向型随机效应的自然选择。

- **算法实现相对可行**：
  - 使用块坐标下降、Newton-Raphson、闭式更新和广义特征分解，适用于中等维度神经影像数据。

- **理论支撑较完整**：
  - 提供了已知方向下的渐近正态性和未知方向下的一致性证明。

- **真实数据应用有科学价值**：
  - HCP Lifespan Study 样本量大，年龄跨度广。
  - 发现的年龄非线性轨迹和晚年方向趋同具有神经科学解释意义。

---

## 8. 不足与局限

- **高斯假设较强**：
  - 模型假设 \(y_{ijt}\) 服从均值为零的多元正态分布。
  - 实际 fMRI 时间序列可能存在非高斯性、自相关、运动伪影、扫描站点差异等问题。

- **方向推断不完整**：
  - 理论上只证明方向估计一致性，没有给出 \(\gamma_i\) 或总体方向 \(\gamma\) 的渐近正态推断。
  - 置信区间主要针对回归系数，未量化脑网络方向本身的不确定性。

- **bootstrap 固定方向可能低估不确定性**：
  - 两阶段 bootstrap 固定 \(\hat{\gamma}_i\)，没有重新估计方向。
  - 虽有渐近依据，但在有限样本或高维情况下，可能低估回归系数的不确定性。

- **vMF 分布是单峰模型**：
  - 如果簇特异方向存在多个模式，例如不同亚型脑老化轨迹，单一 vMF 分布可能不足。
  - 作者也指出未来可考虑球面混合分布。

- **多成分提取为 sequential deflation**
  - 顺序提取计算方便，但可能不如联合多成分估计高效。
  - deflation 和秩补全步骤可能引入额外近似误差。

- **对比实验有限**：
  - 主要与 SCAP 比较，缺乏与更广泛协方差回归方法的系统 benchmark。
  - 难以判断 MCAP 相对所有现有 covariance regression 方法的整体优势。

- **模拟设置偏理想化**：
  - 模拟数据按模型假设生成，对 MCAP 有利。
  - 缺少模型错设、非高斯噪声、簇大小极不均衡、协变量强相关等复杂场景测试。

- **真实数据外部验证不足**：
  - 只在 HCP Lifespan Study 上展示应用。
  - 缺少其他独立生命周期神经影像队列或临床队列验证。

- **稀疏化解释步骤较临时**：
  - HCP 分析中为了解释脑区贡献使用 fused lasso 稀疏化，但该步骤是 ad hoc 的。
  - 稀疏化没有被整合进主模型，也未系统评估其对结论的影响。

- **计算资源未报告**：
  - 虽然给出算法复杂度，但缺乏实际运行时间和硬件信息。
  - 不利于评估方法在更大规模脑区划分或更高维连接矩阵中的实际可扩展性。
