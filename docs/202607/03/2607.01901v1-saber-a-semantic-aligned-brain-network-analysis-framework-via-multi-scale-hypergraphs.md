## 摘要
有效的脑疾病诊断需要脑连接模式与高层语义知识的协同。然而，现有方法大多将大语言模型（LLM）的语义作为辅助特征或监督信号，限制了其在决策中的直接作用，并制约了分类的稳定性和鲁棒性。为克服这一局限，我们提出了一种语义对齐的脑网络框架，主动将LLM衍生的语义融入预测过程。具体而言，首先通过全局自注意力机制融入ROI级语义，以丰富节点表示并提供全脑上下文。随后构建多尺度超图，显式建模功能子网络和多ROI交互，弥补传统图神经网络（GNN）的局部性局限，捕获高阶依赖关系。最后，通过决策级语义对齐机制，将患者特定的文本嵌入选择性注入图表示，使语义能够直接引导预测而不扰动底层网络结构。在公开脑网络数据集ABIDE和ADHD-200上的实验表明，该方法在性能、稳定性和可解释性上均达到当前最优，尤其在小样本场景下表现突出。

## Abstract
Effective brain disease diagnosis requires the synergy of brain connectivity patterns and high-level semantic knowledge. Existing methods, however, largely treat semantics from large language models (LLMs) as auxiliary features or supervision, limiting their direct role in decision-making and constraining classification stability and robustness. To overcome this, we propose a semantic-aligned brain network framework that actively integrates LLM-derived semantics into the prediction process. Specifically, ROI-level semantics are first incorporated via global self-attention to enrich node representations and provide whole-brain context. Multi-scale hypergraphs are then constructed to explicitly model functional subnetworks and multi-ROI interactions, addressing the locality limitations of traditional GNNs and capturing high-order dependencies. Finally, a decision-level semantic alignment mechanism selectively injects patient-specific textual embeddings into graph representations, enabling semantics to directly guide predictions without perturbing the underlying network structure. Experiments on public brain network datasets ABIDE and ADHD-200 demonstrate state-of-the-art performance, enhanced stability, and improved interpretability, particularly in small-sample settings.

## 2. 论文提出的方法论：核心思想、关键技术细节

### 核心思想
通过三个模块的渐进式设计，将语义从节点级注入到图级再到决策级：
- **模块I：多尺度节点级脑网络编码**——注入ROI语义，构建多尺度超图捕获高阶交互。
- **模块II：图级表示抽象**——将多尺度节点表示融合为统一的图级表示。
- **模块III：语义对齐与决策级增强**——使患者特定文本嵌入直接参与分类决策。

### 关键技术细节

#### 模块I：多尺度节点级脑网络编码
- **节点特征初始化**：使用AAL模板将大脑划分为N个ROI，计算BOLD信号间的皮尔逊相关系数，得到初始特征矩阵 X ∈ R^(N×N)。
- **ROI语义嵌入注入**：利用LLM生成每个ROI的详细文本描述（解剖位置、功能角色），通过冻结的预训练文本编码器得到语义嵌入 E_ROI ∈ R^(N×d_LLM)，经线性投影后与原始特征相加，再通过脑网络Transformer（BNT）编码器进行全局自注意力：
  ```
  X0 = BNT( X + E_ROI * Wp )
  ```
  实现全脑语义传播。
- **多尺度超图建模**：在每个尺度k下构建超图 H^(k) = (V, E^(k))，每个超边连接一组语义增强后功能相似的ROI。使用两层超图神经网络（HGNN）进行消息传递：
  ```
  X^(k,l+1) = σ( D_v^(-1/2) * H^(k) * D_e^(-1) * (H^(k))^T * D_v^(-1/2) * X^(k,l) * W^(k,l) )
  ```
  然后通过共享的多头自注意力（MHSA）模块得到尺度特定的节点表示 Z^(k)。

#### 模块II：图级表示抽象
- **池化**：对每个尺度k的节点特征进行全局平均池化得到 g^(k)。
- **自适应多尺度融合**：使用可学习的门控机制计算各尺度权重 α = softmax( f_gate( [g^(1) || g^(2) || ... || g^(K)] ) )，最终图表示 g = Σ α_k * g^(k)。

#### 模块III：语义对齐与决策级增强
- **患者特定语义注入**：引入一个与结构池化解耦的Graph-CLS令牌 q_CLS，与图表示g相加形成条件查询 q = q_CLS + g。该查询通过交叉注意力机制从患者文本嵌入 E_Patient 中选择性提取语义信息：h_fused = CrossAttn(q, E_Patient, E_Patient)。
- **门控残差语义注入**：通过自适应门控机制调整语义注入强度：g' = g + σ(W_g * h_fused) ⊙ h_fused。
- **决策级监督**：使用双重损失优化：
  - 交叉熵损失 L_CE 保证分类精度。
  - 对比损失 L_CL 将g'与LLM生成的疾病嵌入（E_Disease）对齐，拉近同类、推开异类。
  - 总损失 L_total = L_CE + λ L_CL。
