## 摘要
许多复杂系统，如脑网络、金融市场和基因调控回路，并非由固定图描述，而是随时间变化。在每个时刻总结这种结构的标准方法是稀疏精度（逆协方差）矩阵，时变图套索（TVGL）将多变量信号转化为这些矩阵的平滑链。我们引入了TVGL-CFM，这是一个单一模型，能够学习这种链的分布，既可以生成给定类别的新颖、真实的时变网络轨迹，也可以预测观察到的轨迹将如何继续。每个精度矩阵位于正定矩阵的弯曲空间上，但对数欧几里得图将整个轨迹展平为普通向量空间，因此可以在该空间上训练和采样简单的条件流匹配模型，同时每个解码后的矩阵保证是有效的精度矩阵。对于预测，我们不是从噪声开始流动，而是从最近历史的粗略外推开始，因此模型只需学习小的修正。在脑电图运动想象、混沌系统和基因表达数据上，TVGL-CFM生成的轨迹保持了真实数据的类别判别结构，并且比原始信号基线更准确地预测未来的连接性。因此，直接生成结构化的精度轨迹比生成原始信号后估计连接性更为可靠。

## Abstract
Many complex systems such as brain networks, financial markets, and gene-regulatory circuits are described not by a fixed graph but by one that changes over time. A standard way to summarise such structure at each instant is the sparse precision (inverse-covariance) matrix, and the time-varying graphical lasso (TVGL) turns a multivariate signal into a smooth chain of these matrices. We introduce TVGL-CFM, a single model that learns the distribution of such chains and can both generate new, realistic time-varying network trajectories for a given class and forecast how an observed trajectory will continue. Each precision matrix lives on a curved space of positive-definite matrices, but a log-Euclidean chart flattens an entire trajectory into an ordinary vector space, so a simple conditional flow-matching model can be trained and sampled there while every decoded matrix is guaranteed to be a valid precision matrix. For forecasting we start the flow not from noise but from a rough extrapolation of the recent history, so the model only has to learn a small correction. Across EEG motor-imagery, chaotic systems, and gene-expression data, TVGL-CFM generates trajectories that keep the class-discriminative structure of real data, and it forecasts future connectivity more accurately than raw-signal baselines. Generating the structured precision trajectory directly is therefore more faithful than generating raw signals and estimating connectivity afterwards.

## 2. 论文提出的方法论

- **核心思想**：将TVGL估计出的每个时间窗口的稀疏对称正定（SPD）精度矩阵视为位于乘积黎曼流形（Sp⁺⁺）ᵀ上的轨迹。通过全局对数-欧几里得微分同胚将该流形映射到欧氏序列空间，使得标准条件流匹配（CFM）可直接应用于该欧氏坐标，解码后自动得到SPD矩阵。
- **关键技术细节**：
  - **数据预处理**：对多变量信号进行Z-score标准化，窗口化后使用OAS收缩协方差估计，再通过TVGL（ADMM求解）获得稀疏精度矩阵链。
  - **对数-欧几里得嵌入**：对每个窗口的SPD矩阵应用对数-欧几里得图φ(Θ)=vecₗₜ(log Θ)，其中vecₗₜ将下三角部分向量化。整个轨迹变为长度T的向量序列，再进行标准化（减去均值除以标准差）得到最终嵌入。
  - **条件流匹配模型（CFM）**：在嵌入空间训练一个Transformer编码器作为速度场uθ，输入包括当前流动时间s、窗口时间τ和类别标签y。使用线性条件路径z_s = (1-s)z₀ + s z₁，损失为均方速度误差加时间平滑正则项L_temp（对预测终点的窗口间变化施加TVGL同类惩罚）。
  - **预测变体**：将轨迹分为历史（K个窗口）和未来（T-K个窗口）。历史部分通过上下文Transformer编码为条件向量c。未来部分的源分布不再是标准高斯，而是基于历史最近增量估计的随机游走暖启动先验q_rw，然后通过CFM将其修正为真实未来分布。损失添加边界项L_bnd（使预测的第一个窗口与真实历史平滑衔接）。
  - **采样**：使用RK4积分器在流时间s上从0到1积分，得到嵌入终点，再通过逆标准化和矩阵指数解码回SPD精度矩阵。可选的稀疏化对解码后的副对角项进行软阈值后重投影回SPD。

## 7. 优点

- **几何一致性**：利用全局微分同胚将对数-欧几里得嵌入与标准CFM结合，保障每个生成的精度矩阵自动为SPD，无需后验投影。
- **联合建模**：非自回归地一次性生成整个轨迹（或未来块），避免自回归误差累积，且允许未来窗口间相互注意。
- **统一的生成与预测**：同一框架处理两种任务，只需改变条件方式（生成时用类别，预测时用历史+类别）。
- **暖启动策略**：在预测中利用历史信息构造近似源分布，使流专注于学习修正，降低训练难度。
- **时间正则化**：在CFM损失中加入TVGL同类的时间平滑惩罚，有助于保持轨迹的时间一致性。
- **高效性**：单GPU单数据集训练<1小时，采样极快，实用性强。

## 8. 不足与局限

- **TVGL目标质量依赖超参数**：TVGL正则化参数λ和β固定（未按数据集调整），可能限制了目标上限；若参数不当，后续流无法完全补偿。
- **维度缩放困难**：嵌入维度d = p(p+1)/2随通道数p呈二次增长，导致样本复杂度和计算成本急剧增加，限制了高通道系统的应用。
- **稀疏性仅为近似**：通过对数-欧几里得解码后矩阵是稠密的，虽可后处理稀疏化，但会破坏精确零支持，且稀疏化后不一定保持最优几何。
- **局部高斯假设局限**：TVGL依赖局部高斯分布，对强非高斯、重尾数据可能表现欠佳，文中亦承认可替换为其他图估计器（但实验未对比）。
- **实验覆盖不足**：虽然数据种类多样，但缺少大规模真实生物网络（如fMRI静息态高通道脑网络）的验证；基因表达数据使用伪时间构造，可能引入伪迹。
- **偏差风险**：暖启动先验采用简单随机游走，若历史增量估计不准确（如突然结构突变），可能提供错误先验。参数αμ,ασ手动设定，未进行系统调优。
- **对照范围有限**：未与直接在图空间上的黎曼扩散模型（如SPD-DDPM）或图生成模型（如DiGress）进行比较（可能因任务不同）。
