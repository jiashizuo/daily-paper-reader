## 摘要
基于图的功能磁共振成像（fMRI）学习在脑网络分析中展现出巨大潜力。然而，现有方法在跨站点分布外（OOD）设置下性能下降，因为站点条件化混杂因素会诱导非病理性捷径，而通过时间平均构建的功能连接会掩盖瞬时神经动力学，从而限制其对未见站点的泛化能力。本文提出跨站点 OOD 鲁棒脑网络（CORE），这是一个用于跨未见站点脑网络学习的统一框架。CORE 首先执行站点感知的混杂因素解耦，以缓解站点条件化偏差，并提取由可重复诊断性连接边构成的跨站点群体支架。随后，它利用轻量级时间描述符刻画该支架上的瞬时通路动态，并将支架边组织为线图，以实现可迁移的通路层级建模。最后，CORE 引入一种先验引导的受试者自适应门控机制，在利用支架衍生的群体先验的同时保留受试者特异的连接变异性。在真实世界数据集（ABIDE、REST-meta-MDD、SRPBS 和 ABCD）上进行的留一站点交叉评估的广泛实验表明，CORE 始终优于最先进的基线方法，相对提升最高达 6.7%。此外，CORE 对脑图谱变化保持鲁棒，在不同脑分区方案下均维持性能增益。

## Abstract
Graph-based learning on functional magnetic resonance imaging (fMRI) has shown strong potential for brain network analysis. However, existing methods degrade under cross-site out-of-distribution (OOD) settings because site-conditioned confounders induce non-pathological shortcuts, while functional connectivity constructed by temporal averaging obscures transient neurodynamics, limiting generalization to unseen sites. In this paper, we propose Cross-site OOD Robust brain nEtwork (CORE), a unified framework for brain network learning across unseen sites. CORE first performs site-aware confounder decoupling to mitigate site-conditioned bias and extract a cross-site population scaffold of reproducible diagnostic connectivity edges. It then profiles transient pathway dynamics over this scaffold using lightweight temporal descriptors and organizes scaffold edges into a line graph for transferable pathway-level modeling. Finally, CORE introduces a prior-guided subject-adaptive gating mechanism that leverages scaffold-derived population priors while preserving subject-specific connectivity variability. Extensive experiments under leave-one-site-out evaluation on real-world datasets (ABIDE, REST-meta-MDD, SRPBS, and ABCD) show that CORE consistently outperforms state-of-the-art baselines, with up to 6.7% relative gain. Furthermore, CORE remains robust to atlas variations, maintaining performance gains across different brain parcellation schemes.

## 2. 方法论：CORE 框架

论文提出 **CORE：Cross-site OOD Robust brain nEtwork**，由三个核心模块组成：

1. **站点感知混杂解耦**
2. **瞬态通路动态刻画**
3. **先验引导的受试者自适应门控**

整体流程可以概括为：

> 先在训练站点内进行站点特异的混杂去除，提取跨站点稳定的诊断性连接骨架；再在这些骨架边上计算动态 FC 描述符，并构造成线图；最后通过带群体先验的个体化门控进行图消息传递和分类。

### 2.1 站点感知混杂解耦

#### 问题：全局去混杂不足

传统方法往往在所有训练站点上拟合一个统一的混杂回归模型：

\[
\hat r_i^{glob} = r_i - b_{glob} - \Gamma_{glob}^\top \tilde q_i
\]

其中：

- \(r_i\)：受试者的静态 FC 向量；
- \(\tilde q_i\)：标准化后的混杂变量，如年龄、性别、采集参数等；
- \(b_{glob}, \Gamma_{glob}\)：全局截距和混杂回归系数。

论文指出，如果真实生成过程是站点相关的：

\[
r_i = b_e^\star + \Gamma_e^{\star \top}\tilde q_i + s_i^\star + \epsilon_i
\]

那么全局残差仍会保留站点相关偏差：

\[
E[\hat r_i^{glob}|e,\tilde q_i,s_i^\star]
=
s_i^\star + (b_e^\star-b_{glob}) + (\Gamma_e^\star-\Gamma_{glob})^\top \tilde q_i
\]

这说明：只要不同站点的截距或混杂响应关系不同，全局去混杂就会留下残余站点偏差。

---

#### 站点级 Huber 回归

CORE 对每个训练站点 \(e\) 和每条 FC 边 \(j\) 单独拟合 Huber 回归：

\[
(\hat b_{e,j}, \hat \gamma_{e,j})
=
\arg\min_{b,\gamma}
\sum_{i\in D_e}
H_\delta(r_{i,j}-b-\tilde q_i^\top \gamma)
\]

使用 Huber 损失的目的是提升对异常值的鲁棒性。

得到站点去混杂后的残差：

\[
r_i^{site}=r_i-\hat b_e-\hat\Gamma_e^\top \tilde q_i
\]

在未见测试站点上，模型不能利用测试站点标签或重新估计站点特异去混杂器，因此论文将训练站点的去混杂参数做等权平均：

\[
\bar b = \frac{1}{|E_{tr}|}\sum_e \hat b_e,\quad
\bar \Gamma = \frac{1}{|E_{tr}|}\sum_e \hat\Gamma_e
\]

测试时使用冻结的站点无关去混杂器：

\[
r_i^{res}=r_i-\bar b-\bar\Gamma^\top \tilde q_i
\]

### 2.2 跨站点群体连接骨架提取

去混杂后，CORE 进一步从所有 FC 边中筛选出跨站点稳定的诊断性连接边，构成 **population scaffold**。

具体步骤包括：

- 对每个训练站点，分别计算病例组和对照组的鲁棒均值；
- 得到站点级诊断对比：

\[
d_e = \mu_e^{(1)} - \mu_e^{(0)}
\]

- 使用逐边中位数聚合多个站点的诊断对比：

\[
d_{com} = \text{median}_e(d_e)
\]

- 对每条边计算：
  - 诊断对比强度 \(|d_{com,j}|\)；
  - 跨站点符号一致性 \(\kappa_j\)；
  - bootstrap 稳定性 \(\pi_j\)。

最终骨架定义为：

\[
S = \{j: |d_{com,j}|>\tau_E,\ \kappa_j\ge \eta_E,\ \pi_j\ge \zeta_E\}
\]

即只有同时满足诊断效应足够大、跨站点方向一致、bootstrap 稳定的边才被保留。

论文还给出一个 **margin stability** 命题，说明如果去混杂估计误差小于一定边际，经验骨架与理想 oracle 骨架的选择结果一致。

### 2.3 瞬态通路动态刻画

传统静态 FC 使用整段时间序列计算一次相关性，CORE 则只在已经筛选出的骨架边上计算动态 FC，以降低维度和提升可迁移性。

对每个受试者：

- 使用滑动窗口划分 BOLD 时间序列；
- 对每条骨架边 \((u_j,v_j)\)，计算每个窗口内的 Pearson 相关；
- 再做 Fisher-z 变换；
- 得到每条边的动态 FC 序列。

然后用两个轻量时间描述符总结动态轨迹：

1. **时间波动性 volatility**

\[
s_{i,j}
=
\sqrt{
\frac{1}{L_i}
\sum_t
(c_{i,j}^{(t)}-\bar c_{i,j})^2
}
\]

2. **动态灵活性 flexibility**

\[
f_{i,j}
=
\max_t c_{i,j}^{(t)}
-
\min_t c_{i,j}^{(t)}
\]

每条骨架边的初始表示为：

\[
h_{i,p}^{(0)}
=
[
r_{i,j_p}^{site},
\log(s_{i,j_p}+\epsilon),
\log(f_{i,j_p}+\epsilon)
]
\]

也就是说，每条边由：

- 去混杂后的静态 FC；
- 动态波动性；
- 动态范围；

共同表征。

### 2.4 线图建模通路交互

CORE 将“脑区之间的连接边”视为线图中的节点。

- 原始脑图中一条 ROI-ROI 连接，对应线图中的一个节点；
- 如果两条连接共享某个 ROI，则它们在线图中相连。

这样模型不再直接在 ROI 节点上传播，而是在 **通路 / 连接边层面**进行建模。

线图边权由群体诊断先验决定：

- 先根据 \(|d_{com}|\) 得到每条骨架边的 prior score \(\mu_0\)；
- 再根据两个通路 prior score 的接近程度构造线图权重：

\[
A_{pq}
=
\exp
\left(
-\frac{|\mu_{0,p}-\mu_{0,q}|}{\Delta_{\mu_0}+\epsilon_A}
\right)
\]

前提是两条边共享 ROI。

### 2.5 先验引导的受试者自适应门控

为平衡群体稳定模式和个体差异，CORE 为每个受试者、每条骨架通路学习一个 gate：

\[
g_{i,p}
=
\sigma
\left(
\frac{
\psi([h_{i,p}^{(0)}\|c_i])+\lambda \mu_{0,p}
}{\tau}
\right)
\]

其中：

- \(h_{i,p}^{(0)}\)：该受试者在该通路上的局部特征；
- \(c_i\)：受试者整体上下文；
- \(\mu_{0,p}\)：群体骨架先验；
- \(\lambda\)：控制群体先验强度；
- \(g_{i,p}\)：该通路对该受试者的重要性。

消息传递时，邻居通路的信息会被 gate 调节：

\[
h_{i,p}^{(l)}
=
h_{i,p}^{(l-1)}
+
F^{(l)}
\left(
h_{i,p}^{(l-1)},
\sum_q \tilde A_{pq} g_{i,q} h_{i,q}^{(l-1)}
\right)
\]

最后使用 gated readout 得到受试者级表示：

\[
z_i
=
\frac{
\sum_p g_{i,p}h_{i,p}^{(L)}
}{
\sum_p g_{i,p}+\epsilon_z
}
\]

并输入 MLP 分类器。

训练目标为：

\[
L = L_{cls} + \gamma L_{sparse}
\]

其中 \(L_{sparse}\) 用于鼓励 gate 数量接近预算 \(K\)，避免选择过多不稳定通路。

- 部分图表中具体消融数值主要通过柱状图展示，精确数值不够完整；
- 真实临床部署仍需外部独立队列验证。

整体看，实验设计较完整，但在统计检验和可复现实验细节上仍有改进空间。

## 6. 主要结论与发现

论文的主要发现包括：

- CORE 在所有主要跨站点 OOD 数据集上均取得最佳或最强性能。
- 在 ABIDE、REST-meta-MDD、SRPBS、ABCD 等任务上，CORE 相比现有方法有稳定提升，最高相对提升约 **6.7%**。
- 在 ABIDE 的 CC200 脑分区设置下，CORE 仍优于基线，说明其性能不完全依赖 AAL atlas。
- 消融实验表明：
  - 站点感知去混杂显著提升跨站点泛化；
  - 动态通路描述符提供了静态 FC 之外的互补信息；
  - 群体 scaffold prior 和 subject-adaptive gating 都有助于稳定消息传递。
- t-SNE 可视化显示，CORE 学到的表示在 ABIDE 上能更清晰地区分 ASD 与 TD。
- 可解释性分析发现，CORE 识别出的连接涉及：
  - 顶叶和角回；
  - 丘脑-皮层通路；
  - 扣带-岛叶区域；
  - 基底节和纹状体相关结构；
  
  这些区域与既有 ASD 研究中的异常连接模式大体一致。

## 7. 优点与亮点

### 方法层面

- **明确建模跨站点 OOD 问题**  
  将站点视为环境，并围绕站点混杂、动态 FC 和个体差异提出统一框架。

- **站点感知去混杂更符合多中心数据特性**  
  不假设所有站点共享相同混杂效应，而是逐站点估计混杂响应。

- **跨站点 scaffold 设计较有针对性**  
  通过诊断对比、中位数聚合、符号一致性和 bootstrap 稳定性筛选可复现连接，减少对站点伪相关的依赖。

- **动态建模方式轻量**  
  不直接输入完整动态 FC 序列，而是使用 volatility 和 flexibility 描述符，降低维度并适配不同扫描长度。

- **线图建模通路级交互**  
  将连接边作为节点，更贴合“脑连接通路”层面的疾病分析。

- **群体先验与个体门控结合**  
  scaffold prior 提供跨站点稳定性，subject-adaptive gating 保留个体差异。

### 实验层面

- 数据集覆盖多个疾病与人群；
- 使用严格 LOSO 协议；
- 基线较全面；
- 包含 atlas 鲁棒性、消融、敏感性、可解释性和效率分析；
- 资源消耗较低，方法并不依赖大模型或极高算力。

## 8. 不足与局限

### 方法局限

- **依赖观测到的混杂变量**  
  CORE 的去混杂需要年龄、性别、采集描述符等协变量。如果关键混杂变量缺失或记录不一致，效果可能下降。作者也在结论中提到未来要研究 missing-confounder setting。

- **测试站点只能使用训练站点平均去混杂器**  
  对于与训练站点差异很大的目标站点，简单平均的站点无关去混杂器可能不足。

- **scaffold 依赖训练站点的稳定诊断差异**  
  如果训练站点数量少、疾病效应弱或标签噪声大，筛选出的 scaffold 可能不稳定。

- **动态描述符较简单**  
  volatility 和 flexibility 虽然轻量，但可能无法捕捉更复杂的时序状态转移、频域特征或非线性动态模式。

- **二分类任务为主**  
  实验均是二分类疾病识别，尚未验证多分类、连续症状预测或预后预测任务。

---

### 实验局限

- **没有明确统计显著性检验**  
  虽然报告了均值和标准差，但没有系统提供显著性检验。

- **缺少完全外部独立验证**  
  LOSO 是严格评估，但仍局限在给定数据集内部站点。真正跨联盟、跨国家、跨临床流程的独立外部测试仍有必要。

- **ABCD 样本量较小**  
  从大规模 ABCD 中筛选后只使用 425 个样本，可能未充分利用 ABCD 的规模优势。

- **脑图谱鲁棒性测试有限**  
  只额外测试了 ABIDE 的 CC200，尚未系统比较 Schaefer、Power、Brainnetome 等其他 atlas。

- **资源报告不够完整**  
  虽提供 per-epoch 时间和显存，但未说明 GPU 数量、完整训练时长、重复次数等细节。

---

### 应用限制

- fMRI 数据采集成本高、质量控制复杂，CORE 的临床落地仍依赖稳定的预处理流程。
- 跨站点泛化提升并不等同于临床可用，还需要：
  - 更大规模前瞻性验证；
  - 与临床量表和行为指标结合；
  - 可解释结果的神经科学与医学验证；
  - 对不同年龄、性别、种族和设备条件下公平性的系统评估。
